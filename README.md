This project creates an FX (foreign exchange
dashboard that visualizes multiple currencies
and enables easy comparison. We fetch real-time and historical currency data via API from [SNB](https://data.snb.ch) (Swiss National Bank), providing users with actionable insights and trends across different
currencies.
